THE MAGIC 8 BALL ENGINE

Developed by three Montana Code School students, Magic 8 Ball is a create-your-own-adventure template for the classic fortune telling game of the 1950's.

Using a sophisticated randomizing algorithm and well-designed database schema, you can create and store your own personalized categories and jokes.

Magic 8 Ball was developed on the "MERN" stack: MongoDB, Express.js, React.js and Node.js.
